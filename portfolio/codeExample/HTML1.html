DEFINITION:

HTML Stands for : HYPERTEXT MARKUP LANGUAGE

It is the language used to create webpages.
"HYPERTEXT" refers to the hyper links that an HTML page may contain.

FORMAT:

    <HTML>
    <HEAD>
    <TITLE>THIS IS MY HTML PAGE</TITLE>
    </HEAD>
    <BODY>
        Hi...THIS IS MY FIRST HTML CODE
    </BODY>
    </HTML>

EXAMPLE:

    <HTML>
    <HEAD>
    <TITLE>THIS IS MY HTML PAGE</TITLE>
    </HEAD>
    <BODY>
        <p>Hi...THIS IS MY FIRST HTML PAGE</p>
    </BODY>
    </HTML>

OUTPUT:

THIS IS MY HTML PAGE

Hi...THIS IS MY FIRST HTML PAGE
